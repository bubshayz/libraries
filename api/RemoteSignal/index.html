RemoteSignal

A remote signal in layman's terms is simply an object which dispatches data to a client (who can listen to this data through a client remote signal) and listens to data dispatched to it self by a client (through a client remote signal).

note

Argument limitations do apply since remote events are internally used by remote signals to dispatch data to clients.
