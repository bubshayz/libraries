ClientRemoteSignal

The clientside counterpart of RemoteSignal. A client remote signal in layman's terms is just an object connected to a serverside remote signal.

Types​

SignalConnection​

</>
interface SignalConnection {
Disconnect: () → ()
Connected: boolean
}

For more information, see SignalConnection.

ClientRemoteSignal​

ClientRemoteSignal.ClientRemoteSignal: Type

Functions​

is​

</>
ClientRemoteSignal.is(self: any) → boolean

Returns a boolean indicating if self is a client remote signal or not.

connect​

ClientRemoteSignal instance
</>
ClientRemoteSignal:connect(callback: (...any) → ()) → SignalConnection

Connects callback to the client remote signal so that it is called whenever the serverside remote signal (to which the client remote signal is connected to) dispatches some data to the client remote signal. The connected callback is called with the data dispatched to the client remote signal.

connectOnce​

ClientRemoteSignal instance
</>
ClientRemoteSignal:connectOnce(callback: (...any) → ()) → SignalConnection

Works almost exactly the same as ClientRemoteSignal:connect, except the connection returned is disconnected immediately upon callback being called.

disconnectAll​

ClientRemoteSignal instance
</>
ClientRemoteSignal:disconnectAll() → ()

Disconnects all connections connected via ClientRemoteSignal:connect or ClientRemoteSignal:connectOnce.

fireServer​

ClientRemoteSignal instance
</>
ClientRemoteSignal:fireServer(...: any) → ()

Fires ... arguments to the serverside remote signal (to which the client remote signal is connected to).

wait​

ClientRemoteSignal instanceyields
</>
ClientRemoteSignal:wait() → any

Yields the thread until the serverside remote signal (to which the client remote signal is connected to) dispatches some data to this client remote signal. 

-- Server
remoteSignal:fireAllClients("Hi")

-- Client
print(clientRemoteSignal:wait()) --> "Hi"

destroy​

ClientRemoteSignal instance
</>
ClientRemoteSignal:destroy() → ()

Destroys the client remote signal and renders it unusable.
