ClientRemoteProperty

The clientside counterpart of RemoteProperty. A client remote property in layman's terms is just an object connected to a serverside remote property.

Properties​

updated​

This item is read only and cannot be modified. Read OnlySignalClientRemoteProperty instance
</>
ClientRemoteProperty.updated: Signal<newValue: any>

A signal which is fired, whenever the value of the serverside remote property (to which this client remote property is connected) is updated.

Incase the client has a specific value set for them in the serverside remote property, then this signal will only fire if that value has been updated.

ClientRemoteProperty:get() → any

Returns the value of the client stored in the serverside remote property (to which the client remote property is connected to). If there is no value stored specifically for the client, then the serverside remote property's current value will be returned instead.
